# Fisica di Irene 🔬

App di studio per la fisica, **un mattoncino alla volta**. Quattro mattoncini per l'esame, ognuno con lezione + allenamento interattivo + video esplicativi.

- **1 — Le grandezze fisiche** · misurare, unità SI, densità
- **2 — Gli strumenti della fisica** · strumenti, errori, notazione scientifica
- **3 — Descrivere i movimenti** · vettori, velocità, accelerazione, moti
- **4 — Le forze** · forze, legge di Hooke, peso, attrito

App installabile (PWA) su telefono e tablet. Fatto con cura per Irene 🌿
